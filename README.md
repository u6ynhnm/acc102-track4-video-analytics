# Short Video Engagement Analytics Tool
## ACC102 Track4 - Interactive Data Product

### 1. Project Purpose
This interactive tool helps short-video creators analyze video performance, including duration, content type, and audience engagement.

### 2. Data Source
- Data type: Public test CSV data
- Fields: video_id, duration, like, comment, collect, type

### 3. Technical Implementation
- Language: Python
- Framework: Streamlit
- Functions: Data cleaning, transformation, analysis, visualization

### 4. Key Insights
1. Short videos (<15s) have higher engagement
2. Tutorial videos perform better than store visit videos

### 5. Live Tool Link
👉 https://acc102-track4-video-analytics-o9hqo9qta3lizv9q8nptyz.streamlit.app/ 👈

### 6. How to Use
Upload the CSV file, and the tool will automatically generate analysis results and charts.

### 7. Limitations
Small sample size; will expand data in future updates.
